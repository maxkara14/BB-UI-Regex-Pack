**🛠️ BB UI Regex Manager**

*Доступные модули:*

Модули имеют встроенные кнопки копирования промпта. Нажми на иконку 📋 рядом с галочкой и вставь текст в тогла пресета.

* 📱 **tablet (SimsOS):** В конце сообщения будет появляться планшет, который отслеживает состояние вашей персоны (энергия, голод), генерирует красивые мудлеты, а также инвентарь, настроение и даже ленту соцсетей с комментариями.
* 🎙️ **radio (Lycoris FM):** Анимированный виджет радиоплеера. Генерирует кринжовый фристайл-рэп от лица ведущей на уличном сленге. К регексу прикреплено радио, которое можно включить прямо в таверне. Если вы желаете, радио можно подключить любое другое, отредактировав регекс и заменив ссылку в разделе `<audio class="l-audio-player" controls preload="none" src="https://ВАША_ССЫЛКА"></audio>`.
* ⌛ **clocks:** Симпатичная плашка с датой, временем, локацией и погодой, которая красиво открывает каждую новую сцену.
* 🚦 **transitions:** Оформляет текст в красивый контейнер и добавляет набор красивых блоков (таймскипы, фокусы, мысли/воспоминания).
* 🧹 **cleaners:** Набор фильтров. Автоматически скрывает "размышления" нейросети (`<think>`) и вырезает сломанные куски HTML/CSS.

---

⚙️Установка самого расширения:

1. Открой SillyTavern.
2. Перейди во вкладку Расширения (Extensions) -> Установить расширение.
3. Вставь ссылку на этот GitHub репозиторий: `https://github.com/maxkara14/BB-UI-Regex-Pack`
4. Перезагрузи страницу (F5).
5. Перейди во вкладку расширений найди **BB UI Regex Manager** и пользуйся.
