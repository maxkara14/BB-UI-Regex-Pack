**🛠 BB UI Regex Manager**

Доступные модули:

* 📱 **phone:** Интерактивный смартфон с экраном блокировки и выезжающей шторкой для отслеживания потребностей, мудлетов (баффов) и инвентаря. Динамически переключает приложения (новостная лента, посты, личные сообщения, сторис) и показывает умные пуш-уведомления.
* 🎙 **radio (Lycoris FM):** Анимированный виджет радиоплеера. Генерирует кринжовый фристайл-рэп от лица ведущей на уличном сленге. Можно подключить реальный аудиопоток, заменив ссылку в регексе.
* ⌛️ **clocks:** Симпатичная плашка с датой, временем, локацией и погодой.
* 🚦 **transitions:** Оформляет текст в кинематографичные контейнеры. Генерирует красивые названия для эпизодов и добавляет парные блоки (мысли, фокус, флэшбеки).
* 🃏 **scene cards:** Система динамических иллюстраций. Встроенный промпт заставляет нейросеть сгенерировать картинку под настроение текущей сцены (комедия, драма, хоррор или романтика), а регекс стильно оборачивает её в интерактивную 3D-карту. Карты имеют свою "рубашку" и уникальные CSS-эффекты при наведении мыши (взрывы, глитчи, дождь, свечение).

⚠️ Важно: Для корректной генерации самих изображений необходимо установить расширение [sillyimages](https://github.com/0xl0cal/sillyimages).

* 🧹 **cleaners:** Набор технических фильтров. Автоматически скрывает "размышления" нейросети (`<think>`)

*Модули имеют встроенные кнопки копирования промпта. Нажми на иконку 📋 рядом с галочкой и вставь текст в пустой тогл.*

⚙️Установка самого расширения:

1. Открой SillyTavern.
2. Перейди во вкладку Расширения (Extensions) -> Установить расширение.
3. Вставь ссылку на этот GitHub репозиторий: `https://github.com/maxkara14/BB-UI-Regex-Pack`
4. Перезагрузи страницу (F5).
5. Перейди во вкладку расширений найди **BB UI Regex Manager** и пользуйся.
